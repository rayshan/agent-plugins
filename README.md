# agent-plugins

My personal Claude Code plugin marketplace that contains tools to augment AI agents.

## Installation

In Claude Code:
- `/plugin`
- Tab right to Marketplaces
- \+ Add marketplace
- Enter https://github.com/rayshan/agent-plugins
- Restart Claude Code

From command line:

```bash
# Add marketplace
/plugin marketplace add rayshan/ray-agent-plugins

# Install a plugin
/plugin install ray-agent-plugins
```
